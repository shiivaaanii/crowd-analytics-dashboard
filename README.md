# Crowd Analytics Dashboard

A responsive Angular-based Crowd Analytics Dashboard built as part of the Kloudspot UI Engineer test assignment.

The application provides:
- Site-based analytics
- Overview KPIs and charts
- Crowd entries table with pagination
- Centralized analytics context
- Real-time alerts and live occupancy using Socket.IO
- Simulator controls for live data
